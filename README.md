# Interactive Resume Assistant

A build-ready Next.js starter for Igor Yezhov's interactive resume MVP.

## What is included
- Resume dashboard homepage with hero, positioning, experience, projects, and skills
- Recruiter prompt shortcuts and chat panel
- Rule-based grounded chat API with guardrails and lightweight citations
- Canonical resume data files under `data/canonical`
- Retrieval chunk generation utilities and `/api/reindex`
- `/api/chat` and `/api/health` endpoints

## Getting started
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run start
```
